# Angelie CreatorOS Governance

Angelie CreatorOS is stewarded by ANGELIE WORLD HOLDINGS, INC.,
A California C-Corporation that operates as the central licensing, administration, and franchise company.

## Founder

Mingjie Zhai

## Lead Maintainer

The Founder serves as Lead Maintainer unless delegated.

## Maintainers

Maintainers review:

- Pull Requests
- Database Migrations
- Security
- Documentation
- Releases

## Decision Making

Community feedback is encouraged.

Final approval of:

- Database schema
- Security architecture
- Rights engine
- Creator participation system
- Official releases

rests with the Lead Maintainer.
