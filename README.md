# angelie-creator-os
Creator-owned IP Franchise Holdings

Angelie CreatorOS is a rights-management and creator-participation platform.

Angelie CreatorOS is the authoritative internal record of creators, contributions, IP assets, agreements, rights grants, project usage, participation rules, royalty calculations, statements, and approvals associated with the Angelie Universe.

Angelie CreatorOS is managed under Angelie World Holdings, Inc., which is a Creator-First Franchise operating system with three guiding principles:
1. The universe remains under centralized stewardship so films, games, books, and merchandise can be coordinated efficiently.
2. Every creator has a transparent participation agreement that defines credit, compensation, and ongoing rights from the beginning.
3. Every use of every character is tracked through a centralized registry and royalty system so payments and attribution are based on documented data rather than ad hoc negotiations.

Angelie Creator OS is a rights-management and creator-participation platform, that serves both as a character encyclopedia and a world collaboration.

The central rule
This software distinguish five different things:
1. Who originally created something
2. Who legally owns it
3. Who has licensed it
4. Who controls its use within Angelie canon
5. Who receives money when it is used
