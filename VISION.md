# Why Angelie CreatorOS Exists
to build a creator-friendly rights and participation system 
that avoids many of the ownership and attribution problems seen in traditional entertainment pipelines.
As the project grows, this document helps future contributors understand the philosophy behind technical decisions.

# We're building this in the open.
We're not just making Angelie. 
We're building the operating system for creator-owned entertainment.

# What's Private
Angelie screenplay,
story bible,
characters,
artwork,
production documents,
contracts,
financial models.

# What's Open Source
application code,
UI,
database schema,
documentation,
generic sample data,
