# Contributing to Angelie CreatorOS

Thank you for contributing!

Angelie CreatorOS is an open-source platform designed to help creators manage intellectual property, chain of title, licensing, creator participation, and franchise collaboration.

## Before contributing

Please:

- Search existing issues.
- Discuss major features before implementing them.
- Keep pull requests focused.
- Follow the project coding style.

## Pull Requests

Every pull request should include:

- A description of the problem.
- Your proposed solution.
- Screenshots if UI changes.
- Database migration notes (if applicable).

## Code Standards

- TypeScript
- React
- Supabase
- PostgreSQL

## Respect Creator Rights

Do not upload copyrighted Angelie story assets without permission.

Thank you for helping build the future of creator-owned storytelling.
